# Sber
<h1>Краткосрочный прогноз обыкновенных акций ПАО ‘Сбербанк России’</h1>

Описание

Основные параметры:
Горизонт прогнозирования:
до 2-х торговых сессий;
Периодичность составления прогноза (раз в сутки перед началом торговой сессии);
Периодичность - 1 час;

Торговая площадка:
Мосбиржа

Описание прогноза:
Цена открытия
Точки разворота краткосрочных тенденций (цена и время)
Цена закрытия
Предполагаемое изменение цен следующей торговой сессии (направление изменения цены или внутридневная динамика)
