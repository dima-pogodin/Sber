## Краткосрочный прогноз SBER
<i>Внутридневной прогноз цены обыкновенных акций ПАО ‘Сбербанк России’</i>

#### Основные параметры
<ul>
<li>Горизонт прогнозирования до 2-х торговых сессий.</li>
<li>Периодичность составления прогноза (раз в сутки перед началом торговой сессии).</li>
<li>Периодичность - 1 час.</li>
<li>Торговая площадка - Мосбиржа.</li>
</ul>

#### Прогнозируемые значения
<ul>
<li>Цена открытия.</li>
<li>Точки разворота краткосрочных тенденций (цена и время).</li>
<li>Цена закрытия.</li>
<li>Предполагаемое изменение цен следующей торговой сессии (направление изменения цены или внутридневная динамика).</li>
</ul>
